## 🚀 Featured Projects

### ⚓ Reinforcement Learning Environment: Pirate Ship
- **Description:** Development of a custom stochastic marine simulation environment modeling a complex Markov Decision Process (MDP).
- **Technology:** Built under the Gymnasium API using Python, NumPy, and Pygame. Implemented and benchmarked Q-Learning, SARSA, and Monte Carlo algorithms.

### 🏭 Automated Big Data Infrastructure (Madrid Data Lake)
- **Description:** Containerized microservices architecture designed for automated daily ingestion, ETL transformation, and visualization of urban environmental data.
- **Technology:** Event-driven pipeline orchestration with n8n, RabbitMQ distributed queues, persistent S3 storage on MinIO, PostgreSQL, and Apache Superset dashboards.

### 🧬 Metaheuristic TSP Solver & PID Optimization
- **Description:** High-performance optimization engine designed to solve the Traveling Salesperson Problem (TSP) across Madrid regions and adapt controllers.
- **Technology:** Python, NumPy, and Optuna. Implemented Genetic Algorithms featuring adaptive crossover/mutation operators and Two-Opt local search routines.

### 🎰 NoSQL Architecture for Gaming Ecosystems
- **Description:** Migration of relational schemas into optimized distributed NoSQL environments to manage massive user traffic.
- **Technology:** MongoDB (document design), Cassandra cluster (real-time local leaderboards), Redis (vector embeddings and full-text caching), and Neo4j (Graph Data Science algorithms like Leiden and PageRank).
